# Rule_Zero

**The Operating System for Law.**

Rule_Zero is a high-performance, AI-driven legal compliance and civic knowledge platform. It bridges the gap between complex legal text and actionable reality by offering two distinct lenses: a **Civic Encyclopedia** for everyday citizens and a **Compliance Audit Terminal** for tech-focused startups (E-Commerce & Game Dev).

---

## 🚀 Features

*   **Dual-Lens Interface:** Toggle between Citizen Mode (learning rights) and Business Mode (audit & compliance).
*   **AI-Powered Simplification:** Real-time translation of complex legislative text into plain, actionable language.
*   **Persistent Compliance Audits:** Track progress across sessions with database-backed user authentication.
*   **Professional Reporting:** One-click PDF audit generation for investors, board members, and legal teams.
*   **Neo-Brutalist Design:** A bold, high-contrast, and intuitive user interface optimized for speed and clarity.

## 🛠 Tech Stack

*   **Frontend:** Next.js (App Router), React, Tailwind CSS.
*   **Backend:** Next.js API Routes (Serverless).
*   **Database:** PostgreSQL (via Neon) with Prisma ORM.
*   **Authentication:** Firebase Auth (Google Provider).
*   **PDF Generation:** `jspdf` & `jspdf-autotable`.
*   **Deployment:** Vercel.
